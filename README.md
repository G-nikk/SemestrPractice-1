## Запуск проекта

Для запуска проекта выполните следующие шаги:

**1. Клонирование репозитория:**

```
git clone https://github.com/G-nikk/SemestrPractice-1
```
**2. Запуск сервера (Backend, 3000 порт):**

```
cd back
npm install
node server.js
```
**3. Запуск фронтенда (Frontend, 5173 порт):**
```
cd front
npm install
npm run dev
```
